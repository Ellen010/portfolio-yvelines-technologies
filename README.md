## 📌 Project Overview

**Yvelines Technologies Portfolio** is my **personal showcase website** built to demonstrate my **frontend development skills, interactive 3D components, and modern web technologies**.  

The website emphasizes **visual storytelling, smooth animations, and interactive 3D elements**, while maintaining excellent **performance, responsiveness, and maintainability**.  

This portfolio serves as a professional platform to highlight my **technical expertise, project experience, and creative UI/UX abilities**.

---

## ✨ Key Features

- Fully responsive design (desktop, tablet, mobile)  
- Interactive 3D elements powered by Three.js  
- Smooth, timeline-based animations with GSAP  
- Clean, modern UI with Tailwind CSS  
- Optimized performance and fast loading times  
- Scalable and maintainable React component architecture  
- Contact form integration using EmailJS  

---

## 🧩 Tech Stack

### Frontend
- **React 18** — Component-based UI development  
- **Vite** — Fast development server and optimized production builds  
- **TypeScript** — Type safety and scalable codebase  
- **Tailwind CSS** — Utility-first styling  
- **PostCSS & Autoprefixer** — CSS optimization and cross-browser support  

### Animation & 3D
- **GSAP & @gsap/react** — Advanced animations and timelines  
- **Three.js** — WebGL-based 3D rendering  
- **@react-three/fiber** — React renderer for Three.js  
- **@react-three/drei** — Helpers and abstractions for 3D scenes  
- **Leva & Maath** — Interactive 3D controls and math utilities  
- **react-globe.gl** — Interactive 3D globe visualization  

### Utilities & Integrations
- **EmailJS** — Contact form email integration  
- **React Responsive** — Media query handling for responsive layouts  

### Tooling & Quality
- **ESLint** — Code quality and linting  
- **Vite Plugin React** — Fast Refresh and optimized dev experience

## ▶️ Getting Started

```bash
# Clone the repository
git clone https://github.com/Ellen010/portfolio-yvelines-technologies.git

# Navigate into the project folder
cd yvelines-technologies

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
